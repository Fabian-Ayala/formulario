FLEXBOX MODEL

Display: flex

Flex es uno de los nuevos valores HTML5 para propiedad CSS display, que nos permite maquetar nuestras páginas web de una manera mucho más fácil de lo que se hacía con la forma tradicional, en la que utilizábamos propiedades como float, position, entre otras.

En Flex, tenemos un contenedor (contenedor Flex/padre) que es el elemento que contiene la propiedad "display:flex".

Desde ese contenedor podremos especificar la alineación y el tamaño de los elementos que hay dentro (elementos hijos) y distribuir el espacio restante que hay entre estos, todo esto en ambas direcciones, ya sea horizontal o verticalmente. Es decir, podemos distribuir los elementos que contiene el contenedor padre sin poner nada dentro de los elementos hijos.

FLEX-DIRECTION

La propiedad CSS flex-direction especifica cómo colocar los objetos flexibles en el contenedor flexible definiendo el eje principal (Z o Y) y la dirección (normal o al revés).

Flex-direction: RAW

Elementos alineados del primero al último (normal) en el eje X (horizontal)

JUSTIFY-CONTENT

La propiedad CSS justify-content define cómo el navegador distribuye el espacio entre y alrededor de los items flex, a lo largo del eje principal de su contenedor.

justify-content: CENTER

Los items flex son centrados en el eje X.

Justify-content: SPACE-AROUND

Los items flex se alinean uniformemente de tal manera que el espacio entre dos items adyacentes es el mismo. El espacio vacío anterior al primer item y posterior al último item equivale a la mitad del espacio entre dos items adyacentes. Cada elemento posee el mismo espacio alrededor suyo.

Justify-content: SPACE-BETWEEN

Los items flex se distribuyen uniformemente sobre la línea. El espaciamiento se hace de tal manera que el espacio adyacente entre dos items es el mismo. El borde del comienzo principal y el borde del final principal se alinean al ras con el borde del primer y último item respectivamente. Los elementos tienen el mismo espacio entre si salvo al principio y al final que se encuentran pegados al margen.

Justify-content: FLEX-START

Los items flex se colocan comenzando desde el comienzo principal. El margen del primer item es alineado al ras con el borde del comienzo principal de la línea y cada item siguiente es alineado al ras con el precedente.

Justify-content: FLEX-END

Los items flex se colocan comenzando desde el final principal. El margen del último item es alineado al ras con el borde del final principal de la línea y cada item precedente es alineado al ras con el siguiente.
